# 🚕 NYC Taxi Analytics Dashboard

## 📌 Project Overview

This project analyzes NYC Taxi data (12M+ records) using Power BI, Snowflake, and Google Cloud.

## 🔧 Tech Stack

* Power BI
* Snowflake (Data Warehouse)
* Google Cloud Storage
* SQL
* DAX

## 📊 Key Insights

* Peak taxi demand occurs during evening hours
* Credit card payments dominate (~69%)
* Short trips are the most frequent
* Weekdays generate higher revenue than weekends

## 📁 Dashboard Pages

### 1️⃣ Overview Dashboard

* KPI cards
* Trips by hour
* Trips by day
* Trip distribution

### 2️⃣ Customer Behavior Analysis

* Fare vs distance (scatter plot)
* Payment analysis
* Passenger trends

### 3️⃣ Location & Advanced Insights

* Pickup & Dropoff maps
* Heatmap (Day vs Hour)
* Vendor revenue analysis

## 🗺️ Data Source

NYC Taxi Dataset (January 2015)

## 🚀 Features

* Interactive slicers
* Cloud-based data pipeline
* Advanced DAX measures
* Geospatial analysis
